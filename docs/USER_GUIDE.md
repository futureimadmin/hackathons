# User Guide

## Getting Started

### Logging In

1. Navigate to https://app.example.com
2. Enter your email and password
3. Click "Login"

### Dashboard Overview

The home page displays 5 AI systems:
- Market Intelligence Hub
- Demand Insights Engine
- Compliance Guardian
- Retail Copilot
- Global Market Pulse

Click any card to access the system dashboard.

## Market Intelligence Hub

### Generate Forecast
1. Select product or category
2. Choose forecast period (7-365 days)
3. Select model (ARIMA, Prophet, LSTM, or Auto)
4. Click "Generate Forecast"

### View Trends
1. Select category
2. Choose time range
3. View trend analysis and seasonality

## Demand Insights Engine

### Customer Segmentation
1. Choose number of clusters (2-10)
2. Click "Generate Segments"
3. View RFM analysis and segment characteristics

### Price Optimization
1. Select product
2. View current price and elasticity
3. See recommended optimal price

## Compliance Guardian

### Fraud Detection
1. Enter transaction IDs
2. Click "Analyze"
3. View fraud scores and risk levels

### PCI Compliance
1. Select payment records
2. Run compliance check
3. View masked data and violations

## Retail Copilot

### Ask Questions
1. Type your question in natural language
2. Examples:
   - "What are the top 5 selling products?"
   - "Show me low stock items"
   - "Which customers haven't ordered in 30 days?"
3. View results and SQL query

## Global Market Pulse

### Market Trends
1. Select product
2. Choose time range
3. View trend decomposition

### Regional Comparison
1. Select regions to compare
2. View price differences
3. See statistical significance

## Exporting Data

All dashboards support export to:
- PDF
- Excel
- CSV

Click the "Export" button and choose format.
